mvp - minimum viable product.

Tools-
React
express rest api
redis data store
node worker on cron to pull jobs


Notes:-
1. npm init react ->app APPNAME - to create react app
2. code . ->  to open react project in visual studio.
3. rc enter -> to import react and state of the component.
4. npm i node-fetch -> to fetch github api
5. npm install cron -> to schedule the time when the data should refetch.
5. npm install redis -> database to store fetched data from api.
6. npm install -g redis-cli -> to access redis server without installation
7. npm install express -> to to fetch data from redis.
